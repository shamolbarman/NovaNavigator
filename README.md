# NovaNavigator
NovaNavigator: An autonomous agentic web-operating robot powered by Amazon Nova 2, featuring voice-first interaction and Human-in-the-Loop Verification (HiLV) for secure web automation.
### Key Features:
* **Voice-First Interface:** Powered by Amazon Polly for natural dialogue.
* **Autonomous Navigation:** Uses Amazon Nova reasoning to perform web searches and system tasks.
* **HiLV Security:** Human-in-the-Loop Verification ensures the robot asks for permission before any sensitive action.
* **Real-time Status:** A dynamic dashboard built with CustomTkinter.

### Tech Stack:
* **Brain:** Amazon Nova  (via Bedrock)
* **Voice:** Amazon Polly
* **UI:** CustomTkinter & Pygame
* **Automation:** Python Webbrowser & OS integration
